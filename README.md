# Historial de Temperaturas - Unidad 1

## 📌 Descripción

Este proyecto corresponde a la actividad de la Unidad 1 de la asignatura Estructuras de Datos. Consiste en el desarrollo de un programa en Java que permite registrar y analizar las temperaturas diarias de un mes utilizando arreglos.

El sistema procesa los datos ingresados y calcula información relevante como el promedio de temperaturas, así como la identificación de los días más fríos y más cálidos.

## 🎯 Objetivo

Aplicar los conceptos fundamentales de arreglos mediante la implementación de un programa que permita almacenar, procesar y analizar datos de temperatura.

## ⚙️ Funcionalidades

* Registro de temperaturas diarias (30 días).
* Cálculo del promedio de temperaturas.
* Identificación del día más frío.
* Identificación del día más cálido.
* Representación gráfica básica de temperaturas usando texto (barras).

## 🛠️ Tecnologías utilizadas

* Java
* Maven
* Git

## ▶️ Ejecución del proyecto

1. Clonar el repositorio:

   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```

2. Acceder a la carpeta del proyecto:

   ```bash
   cd nombre-del-proyecto
   ```

3. Ejecutar el programa desde tu IDE o usando Maven:

   ```bash
   mvn compile
   mvn exec:java
   ```

## 📊 Ejemplo de salida

El programa mostrará:

* Promedio de temperaturas
* Día con menor temperatura
* Día con mayor temperatura
* Gráfico de barras representando las temperaturas
